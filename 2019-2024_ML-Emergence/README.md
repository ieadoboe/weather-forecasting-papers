## Machine Learning Emergence (2019-2024)

- **Weyn, J.A., Durran, D.R., & Caruana, R. (2019).** _"Can machines learn to predict weather? Using deep learning to predict gridded 500-hPa geopotential height from historical weather data."_ Journal of Advances in Modeling Earth Systems, 11, 2680-2693.
  - Early demonstration of CNNs for weather forecasting; competitive with simple baselines

- **Rasp, S., et al. (2020).** _"WeatherBench: A benchmark data set for data-driven weather forecasting."_ Journal of Advances in Modeling Earth Systems, 12, e2020MS002203.
  - Established standardized benchmark for comparing ML weather models

- **Keisler, R. (2022).** _"Forecasting global weather with graph neural networks."_ arXiv preprint arXiv:2202.07575.
  - Introduced GNNs for global weather forecasting

- **Lam, R., et al. (2023).** _"Learning skillful medium-range global weather forecasting."_ Science, 382, 1416-1421.
  - GraphCast paper: First ML model matching operational NWP skill (ECMWF HRES) for medium-range forecasts

- **Bi, K., et al. (2023).** _"Accurate medium-range global weather forecasting with 3D neural networks."_ Nature, 619, 533-538.
  - Pangu-Weather paper: 3D Earth-specific architectures for weather forecasting; competitive with GraphCast

- **Pathak, J., et al. (2024).** _"Probabilistic weather forecasting with machine learning."_ Nature, 632, 8252.
  - GenCast paper: First ML probabilistic ensemble forecast exceeding ECMWF ENS
