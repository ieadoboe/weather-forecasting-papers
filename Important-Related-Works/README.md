# Related Work

## Parameterization Development (1974)

- **Arakawa, A., & Schubert, W.H. (1974).** _"Interaction of a cumulus cloud ensemble with the large-scale environment, Part I."_ Journal of the Atmospheric Sciences, 31, 674-701.
  - Foundational cumulus parameterization scheme using cloud work function and quasi-equilibrium closure; represents sub-grid convective processes

## Operational NWP & Reanalysis Infrastructure (2015-2020)

- **Bauer, P., Thorpe, A., & Brunet, G. (2015).** _"The quiet revolution of numerical weather prediction."_ Nature, 525, 47-55.
  - Comprehensive review of operational NWP development; documents how incremental advances in physics, DA, and computing created modern forecasting

- **Hersbach, H., et al. (2020).** _"The ERA5 global reanalysis."_ Quarterly Journal of the Royal Meteorological Society, 146, 1999-2049.
  - ERA5 reanalysis dataset (1940-present) that trains virtually ALL modern ML weather models; 31km resolution with 4D-Var DA
